# Dekod

Dekod is a Decoding and Encoding tool written in Python.
It supports various hasing and cipher algorithms.
It is designed to be a lightweight and easy-to-use tool for devs, pentesters, or secuirity professionals.

Dekod has no requirements outside of the standard Python library

### Support
Dekod supports encoding and decoding the following:
 - Base64
 - Hex
 - URL
 - Binary
 - HTML
 - Caesar cipher
 - ROT13
